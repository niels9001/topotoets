# 🗺️ TopoToets – Noord- & Zuid-Holland

Een simpele topo-quiz voor groep 6. Klik op een stip op de kaart en kies de juiste stadsnaam!

## Steden

**Noord-Holland:** Den Helder, Enkhuizen, Haarlem, Amsterdam, Schiphol  
**Zuid-Holland:** Leiden, Den Haag, Delft, Gouda, Rotterdam

## GitHub Pages

1. Maak een nieuwe repository aan op GitHub
2. Push deze bestanden:
   ```bash
   git remote add origin https://github.com/<jouw-gebruikersnaam>/TopoToets.git
   git branch -M main
   git push -u origin main
   ```
3. Ga naar **Settings → Pages**
4. Kies **Source: Deploy from a branch** → **Branch: main** → **/ (root)**
5. Klik **Save**
6. Na een minuutje is de quiz live op `https://<jouw-gebruikersnaam>.github.io/TopoToets/`
